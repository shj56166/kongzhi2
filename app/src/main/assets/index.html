关于项目

这是一款为用户，电子爱好者和开发者设计的SPP经典蓝牙多功能图形化智能APP。不仅提供了直观的继电器控制界面，还集成了强大的底层指令调试工具，并通过后台服务确保蓝牙连接的稳定和定时任务的精准执行。

category

核心功能

  • dynamic_feed

    多路动态控制

    动态添加、编辑和删除控制卡片，独立配置指令，所有配置自动本地保存，提供极高的灵活性。

  • alarm

    精准后台定时

    利用系统AlarmManager，即使App息屏或设备休眠，任务依然会被唤醒并精准触发，确保自动化万无一失。

  • terminal

    底层指令调试

    为开发者设计，支持HEX/文本格式收发原始数据流，是进行蓝牙协议开发与验证的强大工具。

architecture

应用架构

  • 前端界面 (UI Layer):

    由HTML, CSS, JS构建，运行于原生WebView中，实现快速迭代和现代化的设计效果。

  • 原生后端 (Logic Layer - Kotlin):

    由Kotlin驱动，通过前台服务(Foreground Service)确保后台蓝牙连接稳定，并通过广播接收器(BroadcastReceiver)实现精准定时任务。

  • 通信桥梁 (Bridge):

    通过JavascriptInterface (JS->Native) 和 evaluateJavascript (Native->JS) 实现前端与原生后端的高效双向通信。
